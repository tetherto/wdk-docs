# Usage

The WDK Core module is the central orchestrator for your wallet interactions.

Use the guides below to get started:

*   **[Installation](./guides/installation.md)**: Add the package to your project.
*   **[Instantiation](./guides/instantiation.md)**: Initialize the WDK with a seed phrase.
*   **[Register Wallets](./guides/wallet-registration.md)**: Connect specific blockchains (Ethereum, TON, etc.).
*   **[Manage Accounts](./guides/account-management.md)**: Retrieve accounts and check balances.
*   **[Send Transactions](./guides/transactions.md)**: Transfer native tokens.
*   **[Integrate Protocols](./guides/protocol-integration.md)**: Use Swap, Bridge, and Lending protocols.
*   **[Configure Middleware](./guides/middleware.md)**: Add logging and failover protection.
*   **[Error Handling](./guides/error-handling.md)**: Best practices for security and stability.
